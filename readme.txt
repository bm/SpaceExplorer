Brew Crew - Space Explorer

Start Scene: MenuSceneDescription

You'll start in the Main Menu and navigate the menu into either level 1 or level 2 (if you'd like to follow the story then start with level 1). 

The objective is to collect all batteries & oxygen capsules needed in order to continue and find your missing colleague.

Menu Controls:
- Use cursor and mouse buttons to navigate and select menu items
- Press X to exit game

In-Game Controls:
- Press P to bring up pause menu
- Press R to restart game

Mars Surface Character Controls:
- Move using WASD keys
- Adjust third person camera by moving mouse/trackpad
- Hold LEFT SHIFT to run
- Press SPACE to jump

Moon Surface Character Controls:
- Move using the WASD keys
- Look around with mouse controls
